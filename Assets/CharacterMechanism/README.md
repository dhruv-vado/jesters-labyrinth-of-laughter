# Character Mechanism

## All the scripts and their uses are mentioned below

- I didn't use the CameraBobbing.cs because it didn't work properly
- InputManager.cs handles all the incoming input and other scripts call this for input actions.
- InputSystem_Actions.cs is code of key mapping which is done in InputSystem_Actions.inputactions asset.
- MouseLook.cs script is used to move camera as per mouse movement.
- PlayerManager.cs spawns the player and is used to know if player is getting chased.
- PlayerMovement.cs moves the player according to the mapped controls.
